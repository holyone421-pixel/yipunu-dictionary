# Dictionnaire Yipunu-Français

Pipeline automatisé de collecte, traitement et standardisation des tokens yipunu à partir de sources en ligne.  
Exécution 100 % cloud via GitHub Actions.

## Sources
- My Little Word Land (tableau HTML)
- Parlons Yipunu (PDF)
- Thèses HAL (PDF)
- Sorosoro (web brut)
